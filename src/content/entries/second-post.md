# Welcome to Markdown

Markdown is a lightweight markup language, meaning it's readable as plain text and it isn't overly bloated with tags like HTML.

You can do **bold** text like this, and _italics_ like so. Or even _**combining**_ them.
~~Strike through~~ is also a thing ;).

## Lists

In case you need a shopping list, a todo list, or any kind of list really, Markdown has you covered with the following types of lists:

Ordered list.

1. List item 1
2. List item 2

Unordered list.

- List item 1
- List item 2

Nested list.

- List item 1
- - Nested list item 1
  - Nested list item 2

Task list.

- [ ] Pending list item
- [x] Checked list item

### Code blocks

If you are a developer, code blocks are also available in two options:

Inline `code` blocks.
